vscode

markdown

markdown

Linux

  • 查看内存： free -m

  • 查看进程：

    • jps -l

    • ps -aux

    • ps -ef

  • 下载文件夹：sz filename/*

Gitbook

  • gitbook init

  • gitbook serve

  • gitbook build

Hexo

  • 测试：hexo s

  • 部署：hexo clean && hexo g && hexo d

VIM

Kafka

git

  • 项目局部修改提交用户名和邮箱, 解决commit不计入绿色标记问题：

    • git config user.name 你的目标用户名

    • git config user.email 你的目标邮箱名
